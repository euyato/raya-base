bora com a instalação: 

obs: se não tiver baixado as dependências ou não tiver termux ainda, instale pelo site do fdroid e siga os comandos abaixo.

copia e cola o link no chrome e instale:

https://f-droid.org/repo/com.termux_1021.apk

1° termux-setup-storage

copie e cole no termux e de enter e permita o acesso as pastas do celular.

2° vá executando em ordem de um por um e espere o processo de cada um acabar pra começar o outro, clique y e enter quando precisar.

pkg upgrade -y
 
pkg update -y

pkg install nodejs -y

pkg install nodejs-lts -y

pkg install ffmpeg -y

pkg install libwebp -y

pkg install wget -y

pkg install git -y

3° vamos acessar a pasta do bot né colega, se não como que vamos ligar elekk.

cd /sdcard

cd raya-base 

ou 

cd download/raya-base

se aparecer o nome da pasta verdinho em um dos 2 comandos acima é porquê deu bom, agora é só ligar e saborearmos!

npm start

agora bote seu número no campo onde vai pedir e bote o código no seu WhatsApp na parte de dispositivos conectados!
